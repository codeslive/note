### 模板引擎

> 安装ejs

```bash
npm i ejs
```

> 实现功能

- 变量能在html上使用

> 实现代码

- html

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>ejs模板</title>
</head>
<body>
  <h2>我爱你 <%= china %></h2>
  <p><%= weather %></p>
</body>
</html>
```

- js

```javascript
// 安装EJS
//npm i ejs

//导入ejs
const ejs = require('ejs');
const fs = require('fs');

//定义字符串
const china = '中国';
const weather = '今天天气不错';

//声明变量
const str = fs.readFileSync('./ejs.html').toString();

//使用ejs渲染
const result = ejs.render(str, {china:china, weather});

console.log(result);

```
